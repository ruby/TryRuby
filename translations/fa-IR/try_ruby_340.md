خب، آفرین! انجامش دادی، داری متد خودت رو می‌سازی.
با def شروع کردی، بعد اسم متد رو نوشتی، و بعدش هم لیست آرگومان‌هایی که متد بهشون نیاز داره.
دیدی؟ اصلا ترسناک و خطرناک نبود!
کاری که مونده اینه که داخلش کد روبی بنویسیم و آخرش با end تمومش کنیم.

این هم کدش:
    
    def tame( number_of_shrews )
      number_of_shrews.times {
        puts "Tamed a shrew"
      }
    end

> def تو روبی یعنی «تعریف کن» و Leppard هم اسم اون گروهه
